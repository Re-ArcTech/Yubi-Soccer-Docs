# 議事録作成用 辞書

音声文字起こし・口語録音から議事録を生成する際に参照する辞書です。  
AI エージェントはこのファイルを読み込んでから議事録を作成してください。

## よく出る口語表現 → 文章表現

| 口語・文字起こし | 議事録での表現 |
|---|---|
| ケツ叩き | 進捗確認・督促 |
| わちゃわちゃ感 | ハチャメチャ感・カオス感・賑やかさ |
| はちゃめちゃ | カオス・無秩序なアクション |
| むずすぎる | 操作難易度が高すぎる |
| ぶっ込む | まとめて入れる・統合する |
| 詰める | 詳細を決定する・議論を深める |
| 落ちる | 通信が切断する / 話題から外れる |
| ペペペ | 手早く書く（ニュアンスのみ・議事録には書かない） |
| 定例 | 定例ミーティング |
| 全体 | 全員参加のミーティング |
| サブ（MTG） | 担当者限定の小ミーティング |
