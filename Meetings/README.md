# Meetings（議事録）

チームミーティングの議事録を格納します。

## ルール

- **1会議 = 1ファイル** を基本とする
- ファイル名は `YYYY-MM-DD-（トピック）.md`（英小文字・kebab-case）
- 新規作成時は [_template.md](./_template.md) をコピーして使う
- 企画の「軸」や大きな方針転換は [Proposal/](../Proposal/)、[docs/ADRs/](../docs/ADRs/)、[docs/orders/](../docs/orders/) にも反映する

## ブランチ例

```
docs/meeting-2026-06-15-playtest
```
