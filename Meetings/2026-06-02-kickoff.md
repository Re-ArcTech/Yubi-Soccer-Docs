## 3. 現状・問題・選択肢・アプローチ・結果

### 3.1 起動・ロード時間

| | |
|---|---|
| **現状** | 起動時やシーン遷移のローディングが非常に長い。ローディングバーが止まることもある |
| **問題** | 「意味のない待機時間」になっており、プレイヤー体験を損ねる |
| **選択肢** | ① 純粋な高速化　② ローディング中にゲーム世界観の情報を表示 |
| **アプローチ** | 両立できるよう検討。まず実態調査（どのオブジェクト生成が重いか）が必要 |
| **結果** | 未決。次回以降で技術検証 |

### 3.2 操作難易度（「むずすぎる」問題）

| | |
|---|---|
| **現状** | 移動が速すぎてボールと座標がずれる。キック時に他プレイヤーに邪魔される。左手でデバイス操作しながら右手で指操作するのが困難 |
| **問題** | 初心者が楽しめない難易度。得点が生まれにくくゲームが盛り上がらない |
| **選択肢** | ① ボール付近で吸いつき・固定判定を入れる　② ボールを増やして得点機会を増やす　③ 難易度設定を追加 |
| **アプローチ** | ① と ② を組み合わせる方向で検討 |
| **結果** | 採用方針で合意。実装アプローチのモック作成を次アクションとする |

### 3.3 試合テンポ（ゴール後のダウンタイム）

| | |
|---|---|
| **現状** | ゴール後・試合開始時の「初期位置リセット」がない。中央に戻る時間が長い |
| **問題** | 「暇・無駄＝指がしんどい・退屈」につながる |
| **選択肢** | ① スタート位置への自動ワープ　② ゴール後すぐに再開（リスタート短縮）　③ ラスト30秒はボールを大量生成してわちゃわちゃ化 |
| **アプローチ** | ③ のような「盛り上げ演出による時間圧縮」が面白い方向性として浮上 |
| **結果** | ③ は採用検討。① は必須として次アクションに追加 |

### 3.4 MediaPipe（手認識）の精度・カメラ表示

| | |
|---|---|
| **現状** | 試合以外の画面でもカメラが表示されている。WebのUI更新と動作を同時に処理すると挙動が不安定 |
| **問題** | 不要な場所でのカメラ表示がUIを圧迫する。遅延・精度の問題も残る |
| **選択肢** | ① 試合以外では非表示　② 精度向上のための追加学習 |
| **アプローチ** | ① は確定対応。② は要調査 |
| **結果** | ① を採用決定。② は継続検討 |

### 3.5 演出・世界観

| | |
|---|---|
| **現状** | ゴール演出・選手入場・チーム分け演出がない。ルールがどちらのゴールに入れるかわかりにくい |
| **問題** | ゲームの盛り上がりや理解しやすさに欠ける |
| **選択肢** | ① Nintendo Switch Sports / マリオカート風のUI　② リプレイ・ゴール演出の追加　③ 派手なチーム分け演出（高校生クイズ的な） |
| **アプローチ** | Switch Sports を参考にマッチング画面 → 入場 → 試合の流れを整備する方向 |
| **結果** | 方向性として合意。詳細は Proposal 以降で具体化 |

### 3.6 ステージ設計

| | |
|---|---|
| **現状** | ステージに角があり、そこに詰まると脱出できない |
| **問題** | プレイヤーがハマり体験が悪くなる |
| **選択肢** | ① 円形・丸みのあるステージへ変更　② 複数ステージ種類の追加（ゴール4個、動くゴール等）　③ ギミック設置（加速床、ワープ等） |
| **アプローチ** | 全案を Ver.3.0 の検討対象に含める。世界観と合わせて整理 |
| **結果** | 未決。企画フェーズで優先度付けを行う |

### 3.7 Ver.3.0 の企画の軸（最重要議題）

| | |
|---|---|
| **現状** | Ver.2.0 は「サッカーゲーム」寄りの設計だったが、操作難・テンポの問題が残った |
| **問題** | 「純粋なサッカー再現」を目指すと、スマホ指操作の特性と合わない |
| **選択肢** | ① リアルサッカー再現を引き続き目指す　② 「指を使う楽しさ・わちゃわちゃ感」を軸にする　③ ミニゲーム集として別 product 化 |
| **アプローチ** | ② を軸にしつつ、サッカーのフォーマットを土台として活用する |
| **結果** | **② を採用決定**（[ADR-001](../docs/ADRs/ADR-001.md)）。フォールガイズのような「何でもありのわちゃわちゃ感」を目指す |
