# Tasks（タスク管理）

## ルール

- ステータスは `🔲 未着手` / `🔄 進行中` / `✅ 完了` / `⏸ 保留` で管理
- 毎週の定例ミーティングで内容を更新する
- 大きいタスクは担当者がサブタスクに分割してよい
